Congratulations!

You successfully created a solution consisting of 3 projects to help you testing web applications with Selenium and Specification by Example techniques using SpecFlow.

Complete the following steps to get started:

  1. Right-click on the solution
  2. Chose to Restore NuGet Packages
  3. Build the solution to see that it's all working
  4. Download the webdriver(s) you want to use
  5. Put the webdrivers in the WebDrivers directory of the Specs project
ATTENTION: Do not forget to mark the downloaded webdriver executables af Unblocked (right-click the file, Properties > General > Unblock)

Then you are ready to start the real work: adding code to test your first web page.
As an example we will test the SpecByExample Sample Website.

  1. Expand the Pages project and right-click on the ModuleA folder
  2. Choose Add > New item > SpecByExample Page Adapter
  3. As a name you just enter Homepage and press Enter
  4. In the next wizard you enter the url: http://specbyexamplesamplewebsite.azurewebsites.net/ and click Next
    In case you get the error "Cannot load HTML from the specified url", the website is just warming up, wait a second and try again.
  5. For the next steps just accept the defaults by clicking Next
  6. On the last Wizard page also select create a Features file.
  7. Click Finish to close the Wizard

A set of new files is now created in your project:

  • The Pages project contains a new file called Homepage.webmodel.
  • The custom tool created a Homepage.cs from it containing the PageObject implementation for automating the homepage.
  • As well as a a file named HomepageSteps.cs implementating the Given/When/Then steps to arrange, act and assert.
  • In the Specs project a similar folder ModuleA is created
  • Within that folder a specs file is created with HomepageFeature.feature with an example of a scenario (executable as a unit test)

The proof of the pudding is to build the project, reopen the Test Explorer and run MyFirstTest.
This should open a web browser, go to Sample website, then click a random link and check the navigation actually occurred.

This extension can be found under Tools > Extensions and Updates and is named "Specification By Example Extension".
